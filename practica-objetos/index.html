Ejercicios de POO:

Práctica 1

Crear objetos que representen rectángulos. Deben cumplir las siguientes características:

  • Propiedades: ancho y alto
  • Validación en constructor. Comprobar si es un número y mayor que 0, si no, por defecto valor 1.
  • Método cambiarDimensiones: Este método recibe dos números y permite cambiar las dimensiones del rectángulo.
  • Método calcularArea: Este método devuelve el área del rectángulo (ancho x alto)
  • Método copia: Este método devuelve un nuevo objeto que es una copia del rectángulo.
  • Método comparar: Este método recibe otro rectángulo y devuelve:
    • "mayor" si el área del rectángulo actual es mayor.
    • "menor" si es menor.
    • "igual" si las áreas son iguales

Práctica 2

Los objetos de tipo Array (en definitiva, los arrays) poseen numerosos métodos que resultan muy útiles. Esta práctica consiste en añadir un método a todos los arrays para el cálculo de la media aritmética de sus elementos. Modificar el prototipo de los arrays para añadir el llamado método.
Hace falta utilizar el método reduce() de los Arrays.
